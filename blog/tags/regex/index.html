정규 표현식 (Regex)

December 1, 2022 · One min read
Rookedsysc
Just Human

\역슬래쉬​

\d : 숫자
\D : 문자

{숫자}​

숫자만큼 반복
\d{3} : 3자리 숫자

[]

[] 사이의 문자들과 매치

not​

a-z : 문자가 아닌 숫자와 매치
0-9 : 숫자가 아닌 문자와 매치

변수​

() 안에 있는 문자열을 하나의 그룹으로 묶어서 관리
(\d{3})-(\d{4})-(\d{4}) : 010-1234-5678
$1 : 010
$2: 1234
$3 : 5678
