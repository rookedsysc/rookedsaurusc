## container 생성 및 시작
### container 생성

<Tabs>
  <TabItem value="명령어">

  ```bash
  docker run -p 0.0.0.0:22:22 -it -v "[로컬 폴더 주소]":"[도커 폴더 주소]" --name [등록할 컨테이너 이름] [snapshot Image이름] [실행할 bash (ex: /bin/zsh)]
  ```

  </TabItem>
  <TabItem value="예시"> 

  ```bash
  docker run --privileged --platform=linux/amd64 -d -it -v="/Users/rookedsysc/Documents/Github/:/mount" -p 0.0.0.0:80:80 -p 0.0.0.0:4885:443 -p 0.0.0.0:1022:22 -p 0.0.0.0:3000:3000 --name code-server-amd64 code-server-amd64:latest /sbin/init
  ```

  </TabItem>
</Tabs>

### 실행중인 container 접속

```bash
docker exec --privileged -u 1000 -it <docker-name> /bin/zsh
```

## 종료

### 전체 종료

```bash 
docker stop $(docker ps -a -q)
```

## snapshot

### snapshot 생성

<Tabs>
  <TabItem value="How To">

  ```bash
  docker [commit할_container_ID] [snapshot_이름]
  ```

  </TabItem>
  <TabItem value="Example">

  ```bash
  docker commit bdc8d8e96254ebd825f242fd1c4199ed0d1009650bad93393f5116cfff7ad592 code-server-arm:lastest
  ```

  </TabItem>
</Tabs>

### snapshot 목록

```bash
docker images
```

### snapshot 삭제 

```bash
docker rmi <image name>
```
