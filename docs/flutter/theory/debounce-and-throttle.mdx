# Debounce와 Throttle
`debounce`와 `Throttle`은 모두 `특정 함수의 실행을 제한`하는 목적을 두고 있다. <br></br>
`debount`는 특정 기간안의 함수 실행을 모두 취소하고 `마지막에만 실행`한다.<br></br>
`throttle`은 특정 기간안의 함수 실행을 모두 취소하고 `첫번째만 실행`한다.

## Debounce 
지정해놓은 시간 안에 다른 함수가 실행이 되면 `마지막 함수`만 실행이 된다.
