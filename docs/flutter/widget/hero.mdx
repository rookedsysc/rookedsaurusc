# Hero 위젯

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```
Hero 위젯은 같은 태그를 가진 위젯끼리 화면간 이동할 때 그 위젯으로 부터 화면이 `fade in/fade out` 되는 듯한 효과를 준다.

```dart
Hero( 
    tag: Object(heroKey),
    child: image
)
```
