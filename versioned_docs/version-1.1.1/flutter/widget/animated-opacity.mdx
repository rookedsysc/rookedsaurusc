# Animated Opacity

```dart
AnimatedOpacity( // 동작 화면 : (1)
    opacity: _visible ? 1.0 : 0.0,
    duration: const Duration(milliseconds: 500),
    child: Container(
        width: 200.0,
        height: 200.0,
        color: Colors.green,
    ),
);
```
